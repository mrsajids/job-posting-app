# Job Portal App 
A comprehensive job portal application built using Mysql, Express.js, React.js, Node.js.

# installation  
Clone the repo:  

```
git clone https://github.com/exclusiveabhi/react-job-portal.git
```
Install NPM packages:  

```
cd react-job-portal
```
 ```sh
cd backend  
```
 ```sh
npm install  
```  
 ```sh
cd..  
```  
 ```sh
cd frontend
```
 ```
npm install  
```
